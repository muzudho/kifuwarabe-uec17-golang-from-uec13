# run


## ビルド

（PowerShell ではなく、Command Prompt を使って）以下のコマンドを叩いてください。  

```shell
go build
```

`{アプリケーション名}.exe` ファイルが１つ作成されます。  
※ Go言語は、`*.exe` ファイルは１つだけ作成されるのが特徴です。  


## ラン

`{アプリケーション名}.exe` ファイルをダブルクリックしてください。
